## 📖 Що вивчив
<!-- Опиши своїми словами що ти дізнався сьогодні -->
-   розібрався з контролем доступу (Access Control) та статичними властивостями. дізнався, як використовувати private та private(set), щоб захищати дані всередині структури від випадкових змін ззовні. навчився створювати static var та static func для зберігання глобальних даних, які належать усій структурі загалом, а не окремим її екземплярам. 

## 💡 Ключові інсайти
<!-- Що тебе здивувало або що здалося особливо важливим? -->
-   private(set) — це ідеальний інструмент. він дозволяє всім "читати" значення змінної, але право "змінювати" її залишається тільки за самою структурою. це робить код набагато безпечнішим. static круто підходить для створення загальних лічильників (наприклад, загальної скарбнички для всіх крамниць).

## 🤔 Що було складно
<!-- Чесно запиши що було незрозуміло — це допоможе відстежити прогрес -->
-   алгоритмічне мислення — перекладати людські умови (наприклад, перемикання передач у машині з лімітами) на мову логічних операторів if/else всередині методів. але з практикою це стає інтуїтивнішим.

## 💻 Код дня
<!-- Вставити найцікавіший фрагмент коду або посилання на файл -->
```swift
struct MagicShop {
    static var totalGoldEarned = 0
    private(set) var shopGold = 0
    mutating func sellItem(price: Int) {
        shopGold += price
        MagicShop.totalGoldEarned += price
        print("Дзінь! 💰 Товар продано за \(price) золотих!")
    }
}

var magicWorldShop = MagicShop()
magicWorldShop.sellItem(price: 50)
magicWorldShop.sellItem(price: 150)

print("У касі крамниці: \(magicWorldShop.shopGold)") // має вивести 200
print("Загалом мережа заробила: \(MagicShop.totalGoldEarned)") // має вивести 200
```
