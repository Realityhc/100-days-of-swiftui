## 💡 Ключові інсайти
<!-- Що тебе здивувало або що здалося особливо важливим? -->
- мнемоніка для тернарного оператора WTF (What, True, False) — ідеальний спосіб запам'ятати його структуру! також усвідомив різницю між незалежними if та ланцюжком else if — останній гарантує, що спрацює лише одна умова, роблячи код безпечнішим. switch ідеально працює в парі з enum.

## 🤔 Що було складно
<!-- Чесно запиши що було незрозуміло — це допоможе відстежити прогрес -->
- тернарний оператор може бути дуже підступним! потрапив у пастку в тесті, де логічне False виводило рядок "Success", а запитання вимагало підтвердити, чи надрукується це слово. це вимагає максимальної концентрації на тому, що є умовою, а що — результатом.

## 💻 Код дня
<!-- Вставити найцікавіший фрагмент коду або посилання на файл -->
```swift
var AuraHealth = 90
if AuraHealth >= 80 {
    print("Aura повна сил!")    
} else if AuraHealth > 30 {
    print("Aura готова до бою.")
} else {
    print("Обережно, потрібно випити зілля!")
}

enum EnemyType {
    case slime, goblin, dragon
}

let currentEnemy = EnemyType.slime

switch currentEnemy {
case.slime:
    print("Використовуй звичайний удар.")
case.goblin:
    print("Підніми щит!")
case.dragon:
    print("Тікай що є сили!")
}

var mana = 15

let spellResult = mana >= 10 ? "Вогняна куля летить у ціль! 🔥" : "Недостатньо мани 💧"
print(spellResult)
```
