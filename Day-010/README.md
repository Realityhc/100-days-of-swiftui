## 📖 Що вивчив
<!-- Опиши своїми словами що ти дізнався сьогодні -->
-   увійшов в епоху власних типів даних! навчився створювати структури (Structs). розібрався з обчислюваними властивостями (computed properties), де get читає дані за формулою, а set оновлює інші значення. навчився використовувати спостерігачів didSet та willSet для реакції на зміни. освоїв власні ініціалізатори (init) та функції, що змінюють структуру (mutating func).

## 💡 Ключові інсайти
<!-- Що тебе здивувало або що здалося особливо важливим? -->
-   обчислювані властивості get і set працюють як розумний калькулятор із двома кнопками. get рахує значення "на льоту", а set дозволяє змінювати приховані властивості структури через одну контрольну точку. спостерігач didSet — ідеальний інструмент для автоматичних сповіщень (наприклад, коли гравець отримує новий рівень).

## 🤔 Що було складно
<!-- Чесно запиши що було незрозуміло — це допоможе відстежити прогрес -->
-   спочатку було незвично читати формули всередині set і розуміти, як саме працює магічне newValue. здавалося, що воно замінює властивість напряму, але насправді воно бере участь у формулі, яка затирає старі дані.

## 💻 Код дня
<!-- Вставити найцікавіший фрагмент коду або посилання на файл -->
```swift
struct WitchProfile {
    let name: String
    var baseMana: Int
    
    var level: Int {
        didSet {
            print("🎉 \(name) досягла \(level) рівня!")
        }
    }
    
    var totalMana: Int {
        get {
            baseMana * level
        }
    }
    
    // 3. Спеціальний ініціалізатор (Custom Init)
    init(name: String, baseMana: Int) {
        self.name = name
        self.baseMana = baseMana
        self.level = 1 // автоматично ставимо 1 рівень при створенні
    }
    
    // 4. Метод мутації (Mutating Method)
    mutating func levelUp() {
        level += 1
    }
}

// ⚔️ Тестуємо магію:
var aura = WitchProfile(name: "Aura", baseMana: 50)

print("Початкова мана: \(aura.totalMana)") // Виведе 50
aura.levelUp() // Спрацює didSet і виведе: 🎉 Aura досягла 2 рівня!
print("Нова мана: \(aura.totalMana)") // Виведе 100
```
