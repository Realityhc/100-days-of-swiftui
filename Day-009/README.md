# День [9]: [Замикання, передача функцій у функції та контрольна точка 5]

## 📅 Дата: _[10.03.2026]_

## 📖 Що вивчив
<!-- Опиши своїми словами що ти дізнався сьогодні -->
-   здійснив монументальний перехід до вивчення замикань (closures). розібрався, як зберігати функції у змінні, як передавати їх як параметри в інші функції, та що таке trailing closure syntax. освоїв скорочені імена параметрів ($0, $1), що дозволяють писати максимально лаконічний код.

## 💡 Ключові інсайти
<!-- Що тебе здивувало або що здалося особливо важливим? -->
-   зрозумів магію Method Chaining — як можна об'єднувати filter, sorted та map в один безперервний і красивий ланцюжок команд. код стає схожим на конвеєр, де кожен метод виконує свою роботу і передає дані далі.

## 🤔 Що було складно
<!-- Чесно запиши що було незрозуміло — це допоможе відстежити прогрес -->
-   синтаксис замикань спочатку здається дуже незвичним. було складно звикнути до читання функцій, які не мають назви, і розуміти, куди саме повертаються дані через return всередині фігурних дужок.

## 💻 Код дня
<!-- Вставити найцікавіший фрагмент коду або посилання на файл -->
```swift
let spellPower = [15, 50, 8, 85, 30, 12, 65]
let readySpells = spellPower

    .filter { $0 > 20 }
    .sorted { $0 > $1 }
    .map { "Aura кастує заклинання силою \($0)! ✨" }

for spell in readySpells {
    print(spell)
}
```
