## 🤔 Що було складно
<!-- Чесно запиши що було незрозуміло — це допоможе відстежити прогрес -->
- довелося згадати, як працюють складені оператори присвоєння на кшталт +=. спочатку це вилетіло з голови, але насправді це просто зручне скорочення для математичних дій.

## 💻 Код дня
<!-- Вставити найцікавіший фрагмент коду або посилання на файл -->
```swift
var currentXP = 80
let xpToLevelUp = 100
currentXP += 35
let canLevelUp = currentXP >= xpToLevelUp
print("Досвід \(currentXP). Чи може підвищити рівень? \(canLevelUp)")
```

## 📝 Нотатки з обговорення з AI
<!-- Короткий підсумок обговорення -->
