## 💡 Ключові інсайти
<!-- Що тебе здивувало або що здалося особливо важливим? -->
-   some — це обіцянка комп'ютеру, що функція завжди повертатиме один конкретний тип даних, що дозволяє тримати код чистим (особливо це знадобиться в SwiftUI). розширення працюють як плагіни для браузера — додають нові суперздібності без зміни оригінального коду. а розширення протоколів — це можливість роздати готові методи всім класам та структурам, які цей протокол підписують!

## 🤔 Що було складно

## 💻 Код дня
<!-- Вставити найцікавіший фрагмент коду або посилання на файл -->
```swift
protocol Spell {
    var name: String { get }
}

extension Spell {
    func cast() {
        print("✨ ти чаклуєш заклинання: \(name)!")
    }
}

struct Fireball: Spell {
    var name = "Вогняна куля"
}

var fireball = Fireball()
fireball.cast()
```
