## 💡 Ключові інсайти
<!-- Що тебе здивувало або що здалося особливо важливим? -->
- Ілюзія кнопки "Done": вона не є аналогом "Enter" і нічого не "відправляє". розрахунки відбуваються миттєво в реальному часі. кнопка лише змінює стан @FocusState на false, що дає iOS сигнал сховати клавіатуру.
- Магія діапазонів у ForEach: коли ми передаємо жорсткий діапазон (наприклад, 0..<101), нам більше не потрібен id: \.self. SwiftUI автоматично розуміє послідовність чисел і не потребує додаткових "бейджів" для їх розрізнення.
- Різниця між String та Text: String — це просто сирі дані (текст) у пам'яті комп'ютера, а Text — це візуальний компонент SwiftUI, який малює ці дані на екрані.
