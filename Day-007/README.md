## 📖 Що вивчив
<!-- Опиши своїми словами що ти дізнався сьогодні -->
-   здійснив величезне занурення у світ функцій! навчився створювати функції, передавати в них параметри та повертати результати. розібрався з приховуванням міток параметрів за допомогою підкреслення _. познайомився з неймовірно зручним типом даних — кортежами (tuples), які дозволяють повертати кілька значень одночасно.

## 💡 Ключові інсайти
<!-- Що тебе здивувало або що здалося особливо важливим? -->
-   ортежі (tuples) — це справжня знахідка для повернення згрупованих даних (наприклад, статусу і повідомлення) без необхідності створювати складні масиви чи словники. також дізнався, що якщо функція містить лише один рядок коду, слово return можна взагалі не писати — swift сам зрозуміє, що потрібно повернути!

## 🤔 Що було складно
<!-- Чесно запиши що було незрозуміло — це допоможе відстежити прогрес -->
-   величезна кількість нового синтаксису. спочатку було важко зрозуміти логіку символу _ у параметрах функцій, адже він зустрічається в тестах без контексту виклику. відчувається когнітивне перевантаження від обсягу теми.

## 💻 Код дня
<!-- Вставити найцікавіший фрагмент коду або посилання на файл -->
```swift
// 1. Створюємо функцію з "_" (щоб приховати мітку)
// 2. Вказуємо, що вона повертає кортеж (gold та item)
func defeatMonster(_ monster: String) -> (gold: Int, item: String) {
    // 3. Просто повертаємо значення в дужках (Swift сам розуміє, що це return)
    (50, "Зілля здоров'я")
}

// 4. Викликаємо функцію (без слова monster: ) і зберігаємо результат
let loot = defeatMonster("Гоблін")

// 5. Друкуємо результат, дістаючи значення через крапку
print("Aura перемогла! Знайдено \(loot.gold) монет та \(loot.item).")
```
