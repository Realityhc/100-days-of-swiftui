## 📅 Дата: _[2026.03.05]_

## 📖 Що вивчив
<!-- Опиши своїми словами що ти дізнався сьогодні -->
- познайомився з циклами (loops). вивчив цикл for для перебору діапазонів (наприклад, 1...100) та масивів, а також цикл while, який працює, поки виконується певна умова. дізнався, як керувати циклами за допомогою команд break (повна зупинка) та continue (пропуск кроку). успішно пройшов Checkpoint 3 (задача FizzBuzz)!

## 💡 Ключові інсайти
<!-- Що тебе здивувало або що здалося особливо важливим? -->
- for ідеально підходить, коли точно знаєш кількість кроків, а while — коли дія триває до певної події (наприклад, поки здоров'я ворога більше нуля). чітко засвоїв різницю: = це присвоєння (покласти в коробку), == це порівняння (чи вони однакові?), а -= / += — це скорочення для математичних дій.

## 🤔 Що було складно
<!-- Чесно запиши що було незрозуміло — це допоможе відстежити прогрес -->
- цикли while спочатку здавалися дуже заплутаними, було важко відстежити, як змінна оновлюється на кожному кроці. задача FizzBuzz також вимагала уважності: довелося зрозуміти, чому важливо використовувати else if, щоб кожне число отримувало лише одну правильну відповідь.

## 💻 Код дня
<!-- Вставити найцікавіший фрагмент коду або посилання на файл -->
```swift
var slimeHealth = 30
let auraDamage = 10
while slimeHealth > 0 {
    slimeHealth -= auraDamage
    print("aura завдає удару! у слайма залишилося \(slimeHealth) hp")
}

print("слайма переможено! 🎉.")
```

## 📝 Нотатки з обговорення з AI
<!-- Короткий підсумок обговорення -->
